# 🍎 Apple Stock Performance Analysis

* Overview

This project analyzes Apple's stock performance using Microsoft Excel to identify trends in stock prices, trading volume, volatility, and monthly performance. The goal was to practice financial data analysis and communicate meaningful insights through dashboards and visualizations.

---

## Project Objectives

- Analyze historical Apple stock prices
- Identify long-term price trends
- Examine monthly trading volume
- Calculate summary statistics
- Visualize financial data using Excel
- Practice business reporting and data storytelling

---

* Dataset

The dataset contains historical Apple (AAPL) stock market data including:

- Date
- Open Price
- High Price
- Low Price
- Close Price
- Adjusted Close
- Trading Volume

---

* Analysis Performed

- Daily stock price analysis
- Monthly performance analysis
- Trading volume analysis
- Highest and lowest closing prices
- Trend identification
- Summary statistics
- Data visualization using charts

---

* Key Insights

- Apple showed strong long-term growth during the analysis period.
- Trading volume increased during major market events.
- Monthly analysis highlighted periods of increased volatility.
- Excel dashboards made it easier to communicate financial insights visually.
